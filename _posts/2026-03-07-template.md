## The Problem

Binary search trees allow us to store ordered data while maintaining
efficient lookup, insertion, and deletion.

The key invariant:

- Left subtree < node
- Right subtree > node

## Basic Structure

A node contains:

- value
- left child
- right child

```python
class Node:
    def __init__(self, value):
        self.value = value
        self.left = None
        self.right = None
```
